修改Hosts文件加速Github访问

Posted at 2020-01-20 Github Hosts 

前几天访问Github还好好的，昨天夜里就怎么也上不去了，早上爬起来Ping了几个ip丢到hosts就好了。

如果使用Windows系统，hosts文件一般在C:\Windows\System32\drivers\etc路径下，打开之后追加如下内容即可，这些ip的访问速度可能根据你的网络运营商决定，并且可能会发生变化。

另外 githubusercontent.com 不出意外应该早就被屏蔽。

192.30.253.112 github.com
